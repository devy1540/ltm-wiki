## LTM Wiki Long-Term Memory

A persistent, user-owned memory store is available. Resolve it from
`~/.ltm-wiki/config.json` (`defaultStore`) or a local `.ltm-wiki/config.json`.

- Recall relevant memory before answering when prior context may affect the
  result: read `memory/index.md`, then grep `memory/` for the topic.
- Capture durable, low-risk knowledge without an explicit save request; ask first
  for sensitive, personal, ambiguous, or high-volume writes.
- If `sync` is `git`, pull before relying on memory and commit after durable writes.
- Never store secrets, credentials, private keys, tokens, or content the user says
  not to remember.
