# Privacy Policy

LTM Wiki is a local-file toolkit for AI long-term memory stores.

The project itself does not operate a hosted service, collect telemetry, or send memory store contents to a server. Memory contents are written to files selected by the user or by the AI agent running the toolkit.

AI agents, editors, sync tools, Git remotes, cloud storage, Obsidian plugins, or other software used with an LTM Wiki store may process those files under their own policies. Review those tools separately before storing sensitive information.

Do not store secrets, credentials, private keys, tokens, or content that should not be remembered.
